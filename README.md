## 下载

### Docker

```
docker pull apachecn0/ai-blogs-zh
docker run -tid -p <port>:80 apachecn0/ai-blogs-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g ai-blogs-zh
ai-blogs-zh <port>
# 访问 http://localhost:{port} 查看文档
```
